# CIS3250_F25_Sec1_Group3

## Build Instructions
* To compile: make
* To run: ./legacyCalculator
* To remove object files: make clean

## Code Purpose
The purpose of this code is to provide the user with a set of different calculators. The goal is that the calculators can be used for a variety of different mathematical problems (EX. matricies, trigonometry, basic operations, geometry...). 

## Functionality 
Regular Calculator
Basic arethmatic calculations have a print function that will show a visual of the calculations 
* Plus - takes in two numbers and adds them together 
* Minus - takes in two numbers and subtracts one from another
* Multiply - takes in two numbers and multiplies them together
* Divide - takes in two numbers and divides one from another
* Quadratic Formula - takes in three numbers and find their roots
* Print Fibonacci Sequence - takes in a term and prints a histogram of the fibonacci sequence up to that term
* Roots - takes in number and a root value and calculates that root tof the number 

Scientific Calculator
* Power function - takes in a base and a power and returns their result 
* Factorial Series - takes in a number (x) and finds the factorial (x!), which is a product of all positive integers up to x 
* Fibonacci Series - takes in a number of terms (x) and generate the Fibonacci sequence up to that number of terms, where each term is a sum of the two previous ones
* Triganometry calculations - takes in a value x
    * Sine - (sin x)
    * Cosine - (cos x)
    * Tangent - (tan x)
    * Cosec - (cosec x)
    * Sec - (sec x)
    * Cot - (cot x)
* Matrix functions 
    * Matrix sum -  takes in two matricies (of desired size) and adds them together 
    * Matrix product - takes in two matricies (of desired size) and multiplies them together 
    * Matrix transpose - flips a matrix over its diagonal, makes the rows become columns 
* Conversion functions - preforms differenct conversions based on input

Acountant Calculator - takes in a value 
* Show max - find and output the largest value in the dataset
* Show min - find and output the smallest value in the dataset
* Show x-bar - calculate the average 
* Show range - output the difference between the maximum and minimum values in the dataset 
* Show Med - find the middle value of the dataset
* Show Mode - find the more frequently occuring value
* Show value(sort) - outputs the dataset arranged in asending order 
* Set new value - user enters a new value 

Geometry Calculator 
* Area
    * takes in a users choice of shape (rectangle, circle, or triangle)
    * then takes in dimentions based off choice
    * preforms calculations in generic units 
* Perimeter
    * takes in a users choice of shape (rectangle, circle, or triangle)
    * then takes in dimentions based off choice 
    * preforms calculations in generic units 
* Surface Area
    * takes in a user choice of rectangle prism, sphere, or triangle prism 
    * then then takes in dimentions based off choice
    * preforms calculations in generic units 
* Volume
    * takes in a user choice or rectangle prism or sphere
    * then takes in dimentions based off choice
    * preforms calculations in generic units
